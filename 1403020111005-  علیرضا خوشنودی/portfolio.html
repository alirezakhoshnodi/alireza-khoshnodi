توضیحات پروژه: بازی کوییز

توضیحات کوتاهی درباره پروژه اول:در این پروژه با استفاده از زبان برنامه نویسی پایتون یک بازی کوییز طراحی کرده که نحوه کارش اینطور است که سوال هایی از کاربر در دسته های مختلف پرسیده میشود و کاربر باید از بین چهار گزینه یکی را انتخاب کرده و برای هر سوال سی ثانیه زمان دارد و پس از جواب دادن به سوالات امتیاز نهایی او نمایش داده میشود

توضیحات پروژه: حساب بانکی

در این پروژه با استفاده از پایتون برنامه ای طراحی میکنیم که کاربر به صورت انلاین بتواند در بانک حساب جدیدی باز کند و در ادامه بتواند حساب خود را افزایش دهد یا از حساب خود برداشت کند

توضیحات پروژه: سامانه خرید بلیط هواپیما

در این پروژه با استفاده از پایتون برنامه ای طراحی میکنیم که کاربر بتواند بلیط برای سفر خود تهیه کند برای این کار باید برنامه طوری طراحی شود که اول کاربر مشخصات خود را وارد کند بعد تاریخ پرواز خود را تعیین کند و سپس مبلغ را پرداخت کند و بعد رسید بلیط ان برای کاربر نمایش داده شود
